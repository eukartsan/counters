Интервью Упражнение

В этом приложении есть 2 компонента: Counter and App. В приведенных ниже шагах вы сможете изменить и добавить компоненты для изменения функциональности и реализации.

  1. Обновите компонент Counter, чтобы принять onIncrement and onDecrement обратные вызовы в качестве реквизита и гарантируют, что они будут независимо изменять значения счетчика. Каждый обратный вызов должен принимать одно целое значение в качестве параметра, которое представляет собой сумму, увеличивающую существующее значение счетчика.
  2. Перемещение глобального data массив в состояние компонента для App компонент.
  3. Отдайте четвертую Counter компонент и обеспечить, чтобы его значение обновлялось независимо от других.
  4. Создать Total компонент, который должен отображаться ниже Counter компонентов и всегда отображать текущее количество всех Counter значениях.
  5. Сделайте копию Counter компонент, сохраняя оригинал, поэтому вы можете его просмотреть позже. Затем выполните следующие действия:
    1. удалить onIncrement and onDecrement пропсы из (нового) Counter компонента
    2. добавить сингл onChange callback prop, который принимает один целочисленный параметр — новое значение для счетчика.
    3. Обеспечьте все Counter компоненты продолжают обновляться и функционировать независимо после этого изменения.
